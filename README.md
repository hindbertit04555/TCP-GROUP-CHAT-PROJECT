# TCP Group Chat Project

## Overview
This project implements a multithreaded TCP Group Chat application using Java and JavaFX.

## Structure
- Root project → TCPServer
- TCPClient folder → JavaFX client application

## Features
- Real-time communication
- Multiple clients supported
- User list command (allUsers)
- Read-only mode
- Thread-safe server using ConcurrentHashMap

## Technologies
- Java
- JavaFX
- Maven
- TCP Sockets
- Multithreading
